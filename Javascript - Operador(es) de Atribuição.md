# Javascript - Operador(es) de Atribuição

**=**  - Atribuição

**+=** - Atribuição com soma

**-=** - Atribuição com subtração

***=** - Atribuição com multiplicação

**/=** - Atribuição com divisão

**%=** - Atribuição com módulo

```javascript
let b = 3

b += 2 // b = b+2
b -= 2 // b = b-2
b *= 2 // b = b*2
b /= 2 // b = b/2
b %= 2 // b = b%2
```
